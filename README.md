# RISC-V_FLOATPOINT
This is made as a part of RISC-V Hackathon on the Topic FLOATING POINT which includes designing the square root and parameterized matrix multiplication on the Single precision float point inputs.

All the inputs and outputs are in Single-Precision IEEE754 format.
### FLOAT POINT SQUARE ROOT  

We have used Newton-Raphson method to find the square root of the float number.
We have used exception flag which will become high when the input is a negative number.
